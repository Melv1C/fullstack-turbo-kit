# @repo/utils

Shared utility functions and constants.

## Contents

- **math.ts** — Mathematical utility functions
- **constants.ts** — Shared constants across apps

## Usage

```ts
import { add, subtract } from '@repo/utils';

const result = add(5, 3); // 8
```

## Adding Utilities

Create new utility files in `src/` and export them from `src/index.ts`.
