# Backend

API server built with [Hono](https://hono.dev/) — a lightweight, ultrafast web framework.

## Development

```bash
npm run dev
```

Server runs at `http://localhost:3000`

## Build

```bash
npm run build
```

## Tech Stack

- **Hono** — Web framework
- **TypeScript** — Type safety
- **ESLint** — Code linting
